# Bike Shop Analysis

## Description
Le projet **Bike Shop Analysis** est une solution d'analyse de données permettant de suivre et d'optimiser la performance d'une boutique de vélos. Grâce à une combinaison d'outils de gestion et de visualisation, ce tableau de bord interactif fournit une vue complète sur les ventes, la rentabilité et le comportement des clients.

## Objectifs
- **Analyser la performance commerciale** : Suivi du chiffre d'affaires, des bénéfices et des marges.
- **Identifier les tendances temporelles** : Analyse de la saisonnalité et des pics d'activité.
- **Optimiser la prise de décision** : Meilleure allocation des ressources en fonction des tendances de vente.
- **Segmenter la clientèle** : Comprendre la répartition entre clients occasionnels et réguliers.

## Outils et Technologies Utilisés
- **Excel** : Préparation et nettoyage des données.
- **SQL Server** : Stockage et gestion des données relationnelles.
- **Power BI** : Modélisation et visualisation des données via un tableau de bord interactif.

## Fonctionnalités du Tableau de Bord
- **Vue globale des KPIs** : Revenu total, profit, marge bénéficiaire et nombre de transactions.
- **Analyse temporelle** : Evolution des ventes et des profits par mois et par saison.
- **Répartition des ventes par heure et jour** : Identification des périodes les plus rentables.
- **Segmentation des clients** : Comparaison entre clients occasionnels et enregistrés.

## Installation et Utilisation
### 1. Pré-requis
- **SQL Server** installé avec une base de données configurée pour importer les fichiers Excel.
- **Power BI Desktop** pour ouvrir et modifier le tableau de bord.
- **Excel** pour examiner les données sources.

### 2. Étapes d’installation
1. **Importer les données**
   - Charger les fichiers Excel dans SQL Server.
   - Effectuer les transformations nécessaires avec des requêtes SQL.
2. **Connexion Power BI à SQL Server**
   - Charger les données SQL dans Power BI.
   - Appliquer des transformations et créer les visualisations.
3. **Exploration du Tableau de Bord**
   - Filtrer les données par année et saison.
   - Analyser les indicateurs clés pour optimiser les décisions.
